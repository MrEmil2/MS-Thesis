CasJobs_query.txt: CasJobs (SQL) query run on SciServer to indentify spectral data on SDSS database. Outputs CSV table of data file names.

download_sdss.sh: Bash-based retrieval script executed in Windows Subsystem for Linux (WSL). Downloads files directly to computer.

dataloader.m: resamples and normalizes data, then splits into training and testing subsets and saves as datastores in MATLAB.

optionsdefiner.m: defines training parameters for different experiments

testrunner.m: runs the trainnet function and tests the results for each experiment
